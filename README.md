# Projeto &#10003;ToDo App 

Aplicação para o gerenciamento de projetos e tarefas com o objetivo de 
resolver a questão de organização de tarefa de um ou vários projetos

## :computer: Tecnologias usadas:

- `JAVA`
- `MySQL`
- `SWING`
